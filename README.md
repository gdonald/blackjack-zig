# blackjack-zig

Console Blackjack written in Zig

![Blackjack](https://raw.githubusercontent.com/gdonald/blackjack-zig/master/ss2.png)

![Blackjack](https://raw.githubusercontent.com/gdonald/blackjack-zig/master/ss1.png)

## Running

```sh
zig run main.zig
```
